# Feed: Home Assistant - Hidden features

Home Assistant has some features which not many people know about, but they are very handy!

## Tip 1: Enlarge the dashboard edit dialog 

Enlarge the dialog by clicking on the header.
Just like in the animation.

## Tip 2: Dashboard side menu items rearrange

Long-press on the side menu header to activate the mode to rearrange the side menu items.
