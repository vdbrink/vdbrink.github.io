# Feed: Home Assistant dashboard - KNMI Weeralarm NL

<br>
Integreer de KNMI weer alarm data direct op je eigen Home Assistant dashboard.

Het KNMI heeft een [site](https://www.knmi.nl/nederland-nu/weer/waarschuwingen/) waar ze weer waarschuwingen afgeven.
Hoe mooi is het om deze data ook geïntegreerd op je eigen dashboard te tonen!

<img src="/homeassistant/images_weather/knmi_weather_alarm_site.png" alt="KNMI weer waarschuwingen" width="450px">

<em>De weerwaarschuwing website van het KNMI.</em>

Dit is een screenshot van hoe deze data er bij mij op mijn fysieke dashboard uitziet.\
Zodra er een weer alarm actief is wordt de bijbehorende tekst pas getoond en de achtergrond kleur is in de bijbehorende alarm kleur.

<img src="/homeassistant/images_weather/weather_alarm_banner.png" alt="weer alarm" width="450px">

Om dit te realiseren moet je eerst een Webscraper toevoegen om de data van de KNMI site af te halen en om te zetten naar data in twee sensoren.
Daarna kun je zelf kiezen hoe je dit vervolgens wilt presenteren.

Ga naar de pagina (Engelse uitleg) **Home Assistant dashboard: Weather in The Netherlands** naar het hoofdstuk [Weather alarm ](/homeassistant/homeassistant_dashboard_weather_nl#weather-alarm) om te zien hoe je dit zelf kunt realiseren.
Hier staan meerdere voorbeelden hoe je deze data kunt presenteren.

Ik zie graag screenshots tegemoet hoe het weeralarm er bij jullie nu uit ziet!
