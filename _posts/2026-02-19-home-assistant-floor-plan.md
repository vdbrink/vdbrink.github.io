# Home Assistant floor plan

<img src="/homeassistant/images_floorplan/banner.png" alt="floor plan banner" height="100%">

In Home Assistant the number of sensor values grows and the dashboard gets bigger and bigger.
At some point, you can't oversee it anymore.
This is the moment to split screens up per category or per room.

If you still want a quick overview of the statuses, then a floor plan can help!

You can use a digital map from when you bought the house, scan a paper version or create from scratch one by yourself.

You can create a basic 2D version with only the walls with the sensor icons or values on it.
Or if you have plenty of time to spend, you can go all the way down the rabbit hole and create a 3D version with all your furniture in it.

If you want to know more how to start, go to the page [Home Assistant dashboard: Floor plan](/homeassistant/homeassistant_dashboard_floorplan) to see my journey how I created it.
